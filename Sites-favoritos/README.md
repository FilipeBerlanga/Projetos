## 📋 Descrição

Este é um projeto simples e pessoal que apresenta uma página web estática listando meus sites favoritos. A página foi desenvolvida utilizando apenas HTML e CSS, com um design minimalista em fundo preto e texto branco. Cada site favorito é exibido com seu logotipo, um link direto e a fonte da imagem utilizada.

Os sites incluídos são:
- **YouTube**: Plataforma de vídeos
- **Incredibox**: Jogo musical online
- **Steam**: Loja de jogos digitais
- **Conexo**: Jogo de palavras
- **Discord**: Plataforma de comunicação

## 🚀 Como Usar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/FilipeBerlanga/Projetos.git
   cd Projetos/Sites-favoritos
   ```

2. **Abra o arquivo HTML**:
   - Abra o arquivo `FilipeBerlanga_Ag12_DS_I.html` em qualquer navegador web moderno (Chrome, Firefox, Edge, etc.).
   - A página será carregada automaticamente.

3. **Navegação**:
   - Clique nos logotipos ou links para acessar os sites diretamente em uma nova aba.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização (fundo preto, texto branco, layout das imagens)
